# 💰 Smart Financial Dashboard Generator

A simple C program that generates a personal finance dashboard in HTML based on user input. It calculates expenses, savings, and provides basic financial feedback with a visual report.

---

## 📌 Features

- Input monthly income and expenses
- Calculates total expenses and remaining balance
- Computes savings percentage
- Gives basic financial advice (goal-based)
- Generates a complete HTML dashboard
- Includes:
  - Summary cards
  - Progress bar
  - Expense breakdown table
  - Pie chart visualization

---

## 🛠️ How it works

1. Run the program
2. Enter your income and expenses
3. Set a savings goal
4. The program generates a file:  finance_dashboard.html
5. 5. Open it in your browser
